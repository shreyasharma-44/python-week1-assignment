# Week 1: Python Basics – Personal Introduction Program

## Project Overview
A beginner-friendly Python program that collects user's personal information through interactive questions and displays a formatted bio-data profile. This project demonstrates basic Python concepts like variables, input/output, and string formatting.

## What I Learned
- How to use `input()` function to get user input
- How to store data in variables
- How to use `print()` for output display
- How to format strings using f-strings
- Basic Python program structure and syntax
- How to handle multiple user inputs

## Setup Instructions

### Prerequisites
- Python 3.x installed on your system

### Installation Steps
1. Clone or download this repository
2. Navigate to the project directory:
   ```bash
   cd python_assignment
   ```
3. No additional installations required

### Running the Program
```bash
python3 personal_intro.py
```

## How to Use
1. Run the program: `python3 personal_intro.py`
2. Answer the 5 prompts:
   - Your name
   - Your age
   - Your favorite hobby
   - Your city
   - Your favorite food
3. View your personalized bio-data profile
